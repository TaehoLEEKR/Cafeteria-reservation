## 개요
매장 테이블 예약 서비스 구현 (스프링 부트를 활용한 서버 API 기반)

Use : Spring , Jpa, Mysql ,Docker , Redis

Goal : 식당이나 점포를 이용하기 전에 미리 예약을 하여 편하게 식당/점포를 이용할 수 있는 서비스 개발
